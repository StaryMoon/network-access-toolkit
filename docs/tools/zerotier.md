# ZeroTier

> 类型：`mesh-vpn`
> 状态：**推荐**
> 难度：**低到中**

## 一句话

成熟的虚拟局域网工具，适合把分散设备拉到同一个虚拟网络里。

## 适合谁

需要虚拟局域网、多人设备互联、局域网应用访问

## 官方入口

- 官网：https://www.zerotier.com/
- GitHub：https://github.com/zerotier/ZeroTierOne
- 文档：https://docs.zerotier.com/

## 开源情况

核心组件开源

## 成本说明

个人使用可从免费计划开始，团队和高级功能以官方页面为准。

## 优点

- 概念接近虚拟局域网
- 跨平台
- 适合游戏/局域网服务/设备互访

## 缺点

- 管理面和网络模型需要适应
- 在严格网络环境中可能需要额外配置

## 典型场景

- 远程访问局域网服务
- 多人测试局域网应用
- 跨地域设备组网

## 安全提醒

不要把没有鉴权的服务直接暴露到公网。远程访问工具只解决连通性，不替你解决权限、审计和最小暴露面。
