# ngrok

> 类型：`tunnel`
> 状态：**临时演示**
> 难度：**低**

## 一句话

经典开发者隧道工具，适合快速生成临时公网访问地址。

## 适合谁

快速把本地服务给别人看、调试 Webhook

## 官方入口

- 官网：https://ngrok.com/
- GitHub：https://github.com/ngrok/ngrok-api-go
- 文档：https://ngrok.com/docs/

## 开源情况

核心服务不是开源项目

## 成本说明

免费和付费计划差异较大，以官方页面为准。

## 优点

- 上手快
- 适合 demo 和 webhook
- 文档完善

## 缺点

- 长期稳定暴露服务需要付费计划
- 依赖商业服务

## 典型场景

- 演示本地项目
- 支付/机器人/Webhook 回调测试
- 短期共享服务

## 安全提醒

不要把没有鉴权的服务直接暴露到公网。远程访问工具只解决连通性，不替你解决权限、审计和最小暴露面。
