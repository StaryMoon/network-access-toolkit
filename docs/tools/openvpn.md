# OpenVPN

> 类型：`vpn`
> 状态：**传统可靠**
> 难度：**中**

## 一句话

历史悠久的 VPN 方案，生态成熟，文档和企业部署经验丰富。

## 适合谁

机构网络、传统 VPN 场景、需要兼容老系统的环境

## 官方入口

- 官网：https://openvpn.net/
- GitHub：https://github.com/OpenVPN/openvpn
- 文档：https://openvpn.net/community-resources/

## 开源情况

社区版开源

## 成本说明

社区版免费；商业产品以官方页面为准。

## 优点

- 生态成熟
- 兼容性强
- 企业和学校场景常见

## 缺点

- 配置相对繁琐
- 性能和体验通常不如新一代 mesh 工具省心

## 典型场景

- 传统 VPN 接入
- 学校/机构网络
- 兼容老系统

## 安全提醒

不要把没有鉴权的服务直接暴露到公网。远程访问工具只解决连通性，不替你解决权限、审计和最小暴露面。
