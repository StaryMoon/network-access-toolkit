# Mosh

> 类型：`terminal`
> 状态：**弱网神器**
> 难度：**中**

## 一句话

移动 shell，适合弱网和切换网络时保持终端体验。

## 适合谁

网络不稳定、移动网络、长时间 SSH 会话

## 官方入口

- 官网：https://mosh.org/
- GitHub：https://github.com/mobile-shell/mosh
- 文档：https://mosh.org/

## 开源情况

开源

## 成本说明

软件免费；需要服务端安装并开放 UDP 端口。

## 优点

- 弱网体验好
- 网络切换不容易断
- 终端响应顺滑

## 缺点

- 需要 UDP
- 不适合所有安全策略
- 不替代完整 SSH 功能

## 典型场景

- 校园网/移动热点远程终端
- 长时间服务器维护
- 跨网络切换

## 安全提醒

不要把没有鉴权的服务直接暴露到公网。远程访问工具只解决连通性，不替你解决权限、审计和最小暴露面。
